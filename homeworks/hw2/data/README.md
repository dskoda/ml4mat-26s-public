# Dataset notes

The dataset "A compilation of experimental data on the mechanical properties and microstructural features of Ti-alloys" was produced by Salvador et al on the following paper:

Salvador, C.A.F., Maia, E.L., Costa, F.H. et al. A compilation of experimental data on the mechanical properties and microstructural features of Ti-alloys. Sci Data 9, 188 (2022). https://doi.org/10.1038/s41597-022-01283-9

The original data is present on Zenodo at the following link: https://doi.org/10.5281/zenodo.5727736
