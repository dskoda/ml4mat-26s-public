# Datasets

These datasets are used in data visualization and machine learning in the book. Credits for the authors are specified in this README file.

- [Periodic Table](01-DataScience/ptable.csv) - [github.com/robertwb/Periodic Table of Elements.csv](https://gist.github.com/robertwb/22aa4dbfb6bcecd94f2176caa912b952), curated from reference data; factual data not subject to copyright
- [ABO3 perovskites](02-Datasets/perovskites.csv) - A. Emery and C. Wolverton. Sci. Data 4, 170153 (2017).
- [Solubility](03-Features/data/solubility.csv) - Sorkun, M.C., Khetan, A. & Er, S. AqSolDB, a curated reference set of aqueous solubility and 2D descriptors for a diverse set of compounds. Sci Data 6, 143 (2019). <https://doi.org/10.1038/s41597-019-0151-1>
